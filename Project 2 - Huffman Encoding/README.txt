Objective:
To design and implementthe Huffman encoding of English charactres, using a combined list and binary tree data structure. It also reads and compresses an input text file based on the Huffman encoding produced on the input file into a separate text file. A table of the Huffman encoding of characters in the form "character:frequency:encoding" and the calcualted amount of space saved are also provided.
